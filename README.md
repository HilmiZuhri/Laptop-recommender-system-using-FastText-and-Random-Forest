## ğŸ“– Project Description

This project implements a **Laptop Recommendation System** based on **Natural Language Processing (NLP)**. It utilizes **FastText** for text representation (*word embedding*) and **Random Forest** as the classification algorithm.

The system classifies user needs based on textual input and recommends the most suitable laptop category, such as:

* Gaming
* Multimedia
* Office
* Programming
* Workstation

The predicted category is then used to filter and display relevant laptop recommendations.
